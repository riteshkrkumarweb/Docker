# 🚀 Learning Docker

This repository documents my hands-on journey with **Docker** — from the basics of building and running containers to pushing images to **Docker Hub**.

## 📚 What I'm Learning
- **Docker Basics**: Understanding images, containers, and the Docker CLI.
- **Building Images**: Writing Dockerfiles and creating custom images.
- **Running Containers**: Executing applications inside isolated environments.
- **Managing Containers**: Starting, stopping, and removing containers effectively.
- **Docker Hub**: Uploading and sharing images with the community.

## 🛠️ Goals
- Gain practical experience with containerization.
- Learn how to streamline development workflows using Docker.
- Build confidence in deploying applications across different environments.

## 🌐 Next Steps
- Explore Docker Compose for multi-container applications.
- Experiment with container orchestration tools like Kubernetes.
- Apply Docker in real-world projects.
